# FGATE-Nexus（未发布，Alpha 版）

QQ 与 Minecraft 的跨平台交流，基于 Nuxt 3 和 Naive UI 构建。

## ✨ 已实现特性

- **独立可执行文件**（Bun 打包，无需 Node.js）
- **OneBot 适配器**（基础适配与消息处理）
- **跨平台支持**（Linux, macOS, Windows）
- **WebSocket 实时通信**
- **绑定功能**

## 🚀 快速开始

1. 克隆项目

    ```bash
    git clone <repository-url>
    cd FGATE-Nexus
    ```

2. 安装依赖

    ```bash
    bun install
    ```

3. 启动开发服务器

    ```bash
    bun dev
    ```

4. 访问应用

    打开浏览器访问 `http://localhost:3000`
